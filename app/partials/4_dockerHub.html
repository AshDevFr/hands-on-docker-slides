Docker Hub

What is Docker Hub

Docker Hub is a cloud hosted service from Docker that provides an image registery.

Pull Command

Pull an image or a repository from the registry

$ docker pull [OPTIONS] NAME[:TAG]
$ docker pull [OPTIONS] [REGISTRY_HOST[:REGISTRY_PORT]/]NAME[:TAG]

Example: Ubuntu
Pull the version 14.04 of ubuntu 

$ docker pull ubuntu:14.04

IMAGES Command

$ docker images [OPTIONS] [REPOSITORY[:TAG]]

PS Command

List containers

-a, --all Show all containers (default shows just running) 

$ docker ps [OPTIONS]

Example 

$ docker ps -aCONTAINER ID        IMAGE               COMMAND             CREATED             STATUS                      PORTS               NAMES
33077589655c        alpine              "/bin/sh"           7 minutes ago       Exited (0) 28 seconds ago                       alpine-node

START command

Start one or more containers 

$ docker start [OPTIONS] CONTAINER [CONTAINER...]

Options: 

-a, --attach        Attach STDOUT/STDERR and forward signals
-i, --interactive   Attach container's STDIN

COMMIT Command

Commit a container allow you to turn it into a new image. 

$ docker commit [OPTIONS] CONTAINER [REPOSITORY[:TAG]]

Options: 

-a, --author=""     Author (e.g., "AshDev ")
-c, --change=[]     Apply specified Dockerfile instructions while committing the image
--help              Print usage
-m, --message=""    Commit message
